Microsoft Office

¿Qué es Microsoft Office?

Microsoft Office es nuestra plataforma de productividad con tecnología de nube. Con una suscripción a Microsoft 365, puedes obtener: Las aplicaciones de productividad más recientes, como Microsoft Teams, Word, Excel, PowerPoint, Outlook, OneDrive y mucho más.

¿Qué ofrece Microsoft Office?

  • Word, el procesador de textos
  • Power Point, el editor de presentaciones
  • Exel, la hoja de cálculos
